The ORAN SMO CI/CD is structured like this:
--> Helm chart deploys Jenkins.
    --> Contains Jobs, jenkins config in JCasC
	--> Jobs defined in Job DSL in JCasC
		--> They refer to github Pipelines written in Groovy Script.
